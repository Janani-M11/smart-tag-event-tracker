# Smart Tag Event Tracker

This is a simple full-stack project built using React, TypeScript, Node.js, and Express.  
It simulates how IoT or NFC tags send events to a backend server and how those events are displayed in a web dashboard.

## Features

- Add simulated tag events from the frontend  
- Store events using a Node and Express backend  
- View all events in a table  
- View basic statistics such as total events and unique tags  
- REST API communication between frontend and backend  
- Full TypeScript usage  

---

## Tech Stack

Frontend:
- React
- TypeScript
- Vite

Backend:
- Node.js
- Express
- TypeScript

---

## How to Run the Project

### Backend

cd backend
npm install
npm run dev

Backend runs on:
http://localhost:4000

---

### Frontend

cd frontend
npm install
npm run dev

Frontend runs on:
http://localhost:5173
